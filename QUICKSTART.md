# 快速启动指南

## 安装依赖

```bash
npm install
```

## 运行开发环境

```bash
npm run electron:dev
```

这个命令会：
1. 编译 Electron 主进程代码
2. 启动 Vite 开发服务器（React 前端）
3. 启动 Electron 应用窗口

## 功能说明

当前 Demo 包含以下功能：

1. **左侧分类树**
   - 支持多级目录结构
   - 点击分类可以筛选该分类下的聊天记录
   - 点击"全部"查看所有记录

2. **中间聊天记录列表**
   - 显示选中分类下的所有聊天记录
   - 显示记录标题、预览内容和更新时间
   - 点击记录查看详情

3. **右侧聊天详情**
   - 显示完整的聊天对话记录
   - 区分用户和 Assistant 的消息
   - 显示消息时间戳

## 界面特点

- 现代化的三栏布局设计
- 流畅的交互体验
- 响应式滚动和选择状态
- 清晰的视觉层次
